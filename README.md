# MC COVID-19

Diese Mini-Challenge wurde im Rahmen des Praxisportfolio im Modul Data Wrangling der FHNW Studiengang Data Science im HS23 erstellt. Die Aufgabenstellungen sind im PDF Dokument Minichallenge_Covid19 festgehalten. Schwerpunkte dieser MC sind der Umgang Datumswerten, finden und behandeln von Fehlern in Strings, berechnen von Fallzahlen, Anreicherung mit Zusatzinformationen und graphische Darstellung.
